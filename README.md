# Maps

Simple project to test [mapbox](https://www.mapbox.com/).

## Project setup

Create an .env file in the root directory with the following entries:

```javascript
VUE_APP_MAPBOX_TOKEN=MAPBOX PUBLIC TOKEN
VUE_APP_MAPBOX_STYLE_URL=MAPBOX STYLE URL
```
